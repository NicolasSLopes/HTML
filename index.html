Exemplo de Tags HTML

h1

Esta tag é usada para titulos principais ou subtitulos.

p

Esta tag é usada para paragrafos do texto.

body

Esta tag é usada para definir a base visual da pagina.

head

Esta tag serve para os metadados como links e estilos não visiveis na página.

a

Esta tag serve para a criação de hiperlinks.

img

Esta tag serve para adcionar imagem ao site ultilizando o src.

div

Esta tag serve para criar um conteiner ou uma seção (bloco).

button

Esta tag serve para criar um butão clicavel.
